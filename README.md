# Community-Structure

## 職位申請方式

1. 填完官網的表單
2. 加入 Discord 群組
3. 閱讀完欲申請之職位之相關文檔（包括 `定位`、`必要條件` 等）
4. 準備好相關面試資料或是磨練相關能力
5. 於群組中的 `#幹部申請區` 使用指令 `+ca apply <職位名稱>` 進行申請
6. 進行面試
